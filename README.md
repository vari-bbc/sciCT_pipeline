# sciCT_pipeline
A Nextflow implementation of the Data Processing and Analysis section from the paper *Scalable single-cell profiling of chromatin modifications with sciCUT&Tag* (Janssens et al), which incorporates [sciCTextract](https://github.com/mfitzgib/sciCTextract) demux and [souper-star](https://github.com/FredHutch/souper-star/tree/449a120bf02f8453c1e51076f2d0f5edff83fe19).
